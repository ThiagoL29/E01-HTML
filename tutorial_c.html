Estruturas na linguagem C

O que são

Estruturas, também conhecidas como Registros, definem tipos de dados que agrupam variáveis sob um mesmo tipo de dado. A ideia de usar uma estrutura é permitir que, ao armazenar os dados de uma mesma entidade, isto possa ser feito com uma única variável. Por exemplo, se for preciso armazenar a altura, o peso e a idade de uma pessoa, pode-se criar uma estrutura chamada Pessoa e agrupar os dados em um único tipo de dado

  • Exemplo:

typedef struct {

    float Peso;

    int Idade;

    float Altura;

} Pessoa;

Retorno de Structs em Funções

Como uma struct define um tipo de dado, este tipo pode ser retornado em uma função, da mesma forma que ocorre com qualquer outro tipo de dado.

  • Exemplo:

Pessoa SetPessoa(int idade, float peso, float altura) {

    Pessoa P;

    P.Idade = idade;

    P.Peso = peso;

    P.Altura = altura;

    return P;

}

Passagem de Structs por Referência

É passada para a função uma referência da estrutura, sendo possível alterar o conteúdo original usando-se esta referência.

Para passar uma struct por referência, deve-se passar um ponteiro para a struct.

  • Exemplo:

void SetPessoa(Pessoa *P, int idade, float peso, float altura) {

    P->Idade = idade;

    P->Peso = peso;

    P->Altura = altura;

}
